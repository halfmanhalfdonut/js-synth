# Hudba
It's a keyboard!

You can pick the type of wave you want, then use your keyboard to type

Black keys: `w` `e`     `t` `y` `u`     `o`  `p`

White keys: `a` `s` `d` `f` `g` `h` `j` `k` `l` `;` `'`

Octave Up: 'x'

Octave Down: 'z'

## See it live
[Here](https://inexact-scientist.com/keys)

## Running it locally
`npm install`
`npm start`
